Add Twitter feed image media to your sidebars through widgets.

== Description ==

If your currently using Twitget Twitter feed and you would like to display images too from your Tweets, then install this add-on to display in a separate widget. Simple to install.

== Installation ==

1. Upload the `admire-twitget-media` folder to the `/wp-content/plugins/` directory, or search for **Twitget Media** in your WordPress admin area.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Once activated you will see Twitget Media availble in your widgets menu. Simply, drag and drop into your sidebar areas.
1. All set-up is through Twitget plugin that this plugin depends on.
